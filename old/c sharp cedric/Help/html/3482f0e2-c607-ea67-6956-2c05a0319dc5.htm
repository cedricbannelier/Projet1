SubmitVote
Teste si les paramètres d'entrées (choix, idSondage) sont corrects Ne sauvegarde pas en base de données les votes si il y a un cookie qui porte le nom : Vote+idSondage et que le type du sondage est de type cookie Si il n'y a pas de cookies, il est alors créé afin d'empêcher la possibilité de revoter. On envoie un message à l'utilisteur Ne sauvegarde pas en base de données les votes si il y a la même IP en base de données et que le type du sondage est de type IP Si il n'y a pas l'adresse ip, elle n'est pas présente en base de données pour ce sondage, elle est alors stockée Redirige vers Afficheresultat
